# Aplikasi Navigasi Peta Kampus LPKIA

## Deskripsi

Aplikasi Navigasi Peta Kampus LPKIA merupakan program sederhana yang dibuat menggunakan bahasa pemrograman Python. Program ini bertujuan untuk membantu pengguna menemukan jalur terpendek antar gedung di lingkungan Kampus LPKIA.

Aplikasi memanfaatkan struktur data **Graph** untuk merepresentasikan hubungan antar gedung dan algoritma **Dijkstra** untuk mencari rute dengan jarak paling pendek.

## Fitur

- Menampilkan daftar gedung dan jalur
- Mencari rute terpendek antar gedung
- Menghitung estimasi waktu tempuh
- Menampilkan pesan jika gedung tidak ditemukan

## Struktur Data

- Graph
- Priority Queue

## Algoritma

- Dijkstra

## Bahasa Pemrograman

- Python 3

## Data Gedung

- G. Utama
- G. Serba Guna
- G. Kelas
- Kantin
- Parkiran

## Cara Menjalankan Program

1. Pastikan Python telah terinstal pada komputer.
2. Download atau clone repository ini.
3. Jalankan file berikut:

```bash
python "UAS FIK.py"
```

4. Pilih menu yang tersedia pada aplikasi.

## Output Program

Program dapat:
- Menampilkan seluruh gedung beserta jalurnya.
- Menentukan jalur terpendek antar gedung.
- Menampilkan total jarak.
- Menampilkan estimasi waktu tempuh.
